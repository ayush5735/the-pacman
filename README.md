# The Pac-Man Game

A classic **Pac-Man inspired arcade game** developed in **Python** using the **Pygame** library. The game features smooth player movement, intelligent ghost behavior, maze navigation, score tracking, and an engaging retro gameplay experience.

## Features

- Smooth Pac-Man movement
- Multiple Ghosts with different behaviors
- Maze-based gameplay
- Dot collection system
- Life management
- Score tracking
- Colorful retro-style interface
- Optimized game loop using Pygame
- Collision detection
- Cross-platform support (Windows, macOS, Linux)

## Installation

### Clone Repository

```bash
git clone https://github.com/ayush5735/the-pacman.git
```

### Go to Project Folder

```bash
cd the-pacman
```

### Create Virtual Environment

```bash
python3 -m venv venv
```

### Activate Virtual Environment

**macOS / Linux**

```bash
source venv/bin/activate
```

**Windows**

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install pygame
```

### Run the Game

```bash
python main.py
```
